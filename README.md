
# ShowYourFavorateSongs_WEB
> 2019-2 DB 캡스톤 [완성본]
> 
> 노래방을 좋아하는 사람들을 위한 노래방 애창곡 저장+커뮤니티+정보 제공+채팅 기능

## 개발 기간
2019 9월 초~12월 중순

##  개발에 사용한 시스템, 도구
* Spring Boot (JPA)
* MySQL Workbench
* PyCharm
* KakaoMap WEB api
* Google SMTP
* Cos.jar
* WebSocket
